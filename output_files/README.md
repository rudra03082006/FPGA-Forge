# Output Files

This folder contains compiled FPGA programming files generated from the Quartus project.

These files allow the FPGA to be programmed directly without recompiling the design.

## File Types

- `.sof` – SRAM Object File used to program the FPGA through JTAG.
- `.pof` – Programming Object File used for configuration devices (if required).

Participants can use these files to quickly load the design onto the FPGA board during the workshop.
